# 🎮 Time Rewind Puzzle Game (Unity)

A system-driven puzzle game built in Unity where the player can rewind time and collaborate with their past selves to solve challenges.

---

## 🚀 Core Concept

The player can record their actions and rewind time to spawn a clone that replays those actions.

Using this mechanic, the player must:
- Be in multiple places at once
- Coordinate across timelines
- Solve puzzles involving switches, doors, and moving platforms

---

## 🧠 Features

- ⏪ Time Rewind System  
  Records player movement and replays it through clones

- 👥 Clone-Based Gameplay  
  Past versions of the player assist in solving puzzles

- 🧩 Progressive Level Design  
  - Level 1: Introduces rewind mechanic  
  - Level 2: Multi-switch coordination  
  - Level 3: Timing-based moving platform  

- 🎮 Basic Game Flow  
  - Start screen  
  - Level progression  
  - Level complete screen  
  - Restart functionality  

- 📊 Clone Limitation System  
  Restricts number of clones to increase challenge

---

## 🛠 Tech Stack

- Unity (Game Engine)
- C# (Gameplay Programming)
